# Employee Support System
<h3> Employee Support System is a simple web system for managing support tasks. It helps admins, support staff, and employees handle requests, assign tasks, and track priorities easily. Built with JSP-Servlet, Hibernate, and a user-friendly design. </h3>
<p> - Admin: Manage the system and users, create accounts and assign roles, view and assign employee requests to support staff, and search requests by time range or priority. </p>
<p> - Support Staff: Handle assigned requests, search requests by time range or priority, and update personal account information. </p>
<p> - Employee: Submit requests with chosen priorities, view and search their submitted requests, and update personal account information. </p>

 ## Technology
- Technologies Used: JSP, Bootstrap 5
- Database: Hibernate, MySQL
- Web Server: Apache TomCat Server
